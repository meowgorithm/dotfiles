# The Meowgorithm Dotfiles, Collectors' Edition

This repo contains configuration for:

- macOS aarch64
- Linux x86_64 (Fedora, NixOS, and others, including WSL)

## Getting Started

```bash
# Link all configs
./setup link

# Update dependencies (installs Nix, Homebrew packages, etc.)
./setup update

# Remove all symlinks
./setup remove
```
